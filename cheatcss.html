CSS Section

What is CSS?

  • CSS stands for Cascading StyleSheet
  • Allows developers to create rules that specify how the content of an HTML element should appear on the web page.

    • CSS Syntax

  • CSS works by connecting rules with HTML elements

    • Why use CSS?

  • CSS is used to define styles for web pages
  • CSS allows HTML to be more content focused by removing the style formatting from HTML page
  • It also saves a lot of work!!
    • Stylesheets
  • External Stylesheets are the most preferred way of styling HTML documents
    • Link Attributes
  • "rel": Specifies the relationship between the HTML page and the file it is linked to.
  • "type": Specifies the type of the document being linked to
  • "href": Specifies the path or location to the CSS file. Often the CSS file is placed within a folder called styles.

    • CSS Box Model

  • The box model allows us to add a border around elements, and to define space between elements.
